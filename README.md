# README


##product
  |Column   |Type     |Options |
  |---------|---------|--------|
  |title    |string   |NOT NULL|
  |category |text     |NOT NULL|
  |price    |integer  |NOT NULL|

  belongs_to :user
  has_many   :comments
  has_many   :status
  has_many   :delivery_charge
  has_many   :from
  has_many   :burden

##status

belongs_to :product

##from

##esd

##burden
